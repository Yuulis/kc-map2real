# 当サイトについて
ブラウザゲーム[「艦隊これくしょん -艦これ-」](https://www.dmm.com/netgame/feature/kancolle.html)内の海域マップを実際の地図上に再現したWebアプリです。

ゲームの世界観を史実の地理と結びつける資料として活用できます。

# ライセンス

本サイトで使用している画像は、著作権法第32条に基づき引用しています。著作権はすべて原著作権者様に帰属します。権利者様からの削除依頼や警告には速やかに対処いたします。

# プライバシーポリシー

当サイトでは、アクセス解析のために [Google Analytics](https://developers.google.com/analytics?hl=ja) を使用しています。Google Analytics はデータ収集のために Cookie を使用しますが、収集されるデータは匿名であり、個人を特定するものではありません。

Cookie の使用を望まない場合は、ブラウザの設定から無効にできます。

Google のデータ収集の仕組みについては、[Google のポリシーページ](https://policies.google.com/technologies/partner-sites)をご覧ください。
